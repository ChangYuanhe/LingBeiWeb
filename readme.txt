设计理念：我们认为游戏是一件很酷的事情，在设计网页的时候我们嵌入了大量的特效，希望能够让网页酷一点，更加符合游戏网页的特点。
关于风格：主题由黑神话：悟空延申而来，因此基于它的风格，我们的网页整体呈现为暗色系的风格，这样看起来很酷。

一、Animation页：
我们的网页由Animation.html开始。
内容如其名：
首先，我们在Animation页中设置了一个3D立方体盒子。在立方体的每一个面上都有着游戏的图案，立方体会自动进行旋转，当你将鼠标放置于立方体上时，立方体的六个面将会分散开来并且加快旋转速度。我们设置3D的盒子的目的是粗略介绍游戏，更多的是用一个炫酷的方式来展现游戏，让进入我们网页的用户有意愿继续浏览网页。
其次，我们在3D盒子下设置了Let‘s play的文本，用户将鼠标置于其上时文本框将发出高亮，点击文本框将跳转到index页。
并且，我们在整套网页中都改变了鼠标的样式，希望能够更符合游戏主题。


二、Index
我们的网页由四个部分组成，分别为：游戏介绍，论坛中心，关于我们。在主页中我们将这四个部分平均分配到了页面中，每个部分由小背景图，文字组成。文字为白色，但是随着时间的推移，文字的每一部分将逐渐转为炫彩再转回白色。

在四个小部分的背后，大背景中也加入了炫酷的特效：
1.粒子特效：我们设置了大量的高光粒子，他们会在鼠标移动的时候跟随其移动，在鼠标静置后逐渐聚合于鼠标处，而每当鼠标点击时，粒子都会炸裂开来，像烟花一样四处扩散。
2.星河特效：在背景的四周，我们设置了不断旋转的星河。  


三、游戏介绍
在游戏介绍中，我们希望通过一张吸引人的图片与一句话来安利一款游戏，希望能用这一图一句触动你的内心。
在页面头部，我们设置了跳转模块，可以帮助你快速从游戏介绍部分进入其他部分。当鼠标放置于每个文本框上时有动画效果。

在页面主体，我们设置了8个游戏部分，每个部分由一张图片，游戏名称及一句话组成。
你可以通过鼠标滚轮或者拖动页面右侧的拖动条来上下移动，当鼠标置于某一部分上时，这个部分将会变暗，这看起来很酷。每一个游戏部分的宽度恒定，但是图片的大小比宽度大得多，所以你可以滑动滚轮来一览这张图片的全貌，这也为该页面带来了一些趣味性。
                           

四、论坛中心
进入论坛中心，你将会看到一个炫酷的旋转木马特效，在旋转木马上我们放置了一些极具吸引力的图片（图片采用暗黑系风格——还是为了炫酷），旋转木马下方设计了点击登录按钮，点击后会跳转到登陆页面。
登陆页面希望你输入账号密码，下方有登录按钮，右侧有动态动漫角色。


五、关于我们
点击进关于我们的页面，会在屏幕中下方显示”关于我们“四个大字，下面三行是对我们最好的描述，即”我们是熬夜的秃子！“。
拖动页面到下一部分，左侧有”了解我们“的闪光字，右侧有4个卡片，卡片上写着我们的自我介绍，拖动卡片能够浏览我们每个人的自我介绍。
